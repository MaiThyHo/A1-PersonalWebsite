# Personal Website

This is a personal portfolio website showcasing my background, experience, education, and skills.  
It also includes a **Guess Password** feature for fun user interaction.

---

## How to Run the Website

1. Download or clone the repository to your local computer.
2. Make sure all files are in the same folder structure as provided (HTML, CSS, JS, and assets).
3. Open the `index.html` file in any modern web browser (e.g., Chrome, Firefox, Edge).
4. Navigate through the pages using the navigation menu.

No build process or server setup is required, this website runs entirely in the browser.
However, there are dockerfile and docker-compose.yml for you to run it in localhost:8080

---

## Pages & Features

- **Home Page**  
  Introduction and quick navigation to other sections.

- **About Page**  
  Details about my professional experience, education, and skills.

- **Guess Password Page**  
  Interactive form where the user can input preferences and generate a custom password.

- **Responsive Design**  
  Works on desktop, tablet, and mobile screens using CSS Flexbox.

---

## Additional Notes

- Written in **HTML**, **CSS**, and **JavaScript** only.
- Assets (images, video) should remain in the `assets/` folder for the website to display correctly.
- Styling uses custom CSS; font choices and background images can be changed in `styles.css`.
